# @rn-vui/carousel

Carousels show a collection of items that can be scrolled on and off the screen.

## Installation

```sh
npm install @rn-vui/carousel
```

## Usage


```js
import { multiply } from '@rn-vui/carousel';

// ...

const result = await multiply(3, 7);
```
